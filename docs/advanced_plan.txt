MiniOS Advanced Development Plan
================================

Goal
----
Build MiniOS into a complete custom OS stack with:
- Custom bootloader pipeline (BIOS stage1 + stage2 + payload handoff)
- 64-bit kernel with efficient event loop
- Input drivers (keyboard + mouse)
- Practical app runtime with isolated app services
- Storage, memory, and process subsystems
